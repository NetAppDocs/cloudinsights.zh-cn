== 安装

有关 Kubernetes 安装说明，请参见 link:task_config_telegraf_agent_k8s.html["Kubbernetes监控操作员"] 页面。

Kuber-netes Monitoring Operator会自动安装Kube-state-metrics；无需用户交互。
