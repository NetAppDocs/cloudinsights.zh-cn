= 适用于 ONTAP 9.9+ 的 NetApp 云连接数据收集器
:allow-uri-read: 


[role="lead"]
此数据收集器可创建云连接，以支持从 ONTAP 9.9+ CVO ， AFF 和 FAS 系统收集数据。


NOTE: 自2023年4月4日起、此数据收集器不再可安装在Cloud Insights 中、并将于2023年5月25日从所有Cloud Insights 安装中删除。有关过渡到基于AU的数据收集的信息、请参见 link:https://kb.netapp.com/Advice_and_Troubleshooting/Cloud_Services/Cloud_Insights/How_to_transition_from_NetApp_Cloud_Connection_to_AU_based_data_collector["知识库"^]。
